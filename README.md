## 💡功能列表

* [X] 自动获取游戏窗口
* [X] 自动跳过 / 连播剧情
* [X] 自动切换章节

## 📕使用方式

1. 准备好Python运行环境 推荐版本 `3.8`
2. 在本项目目录下执行 `pip install -r requirements.txt` 安装所需依赖
3. 使用任意方式(如: scrcpy)将您的游戏窗口投屏到电脑，并且能够鼠标点击操作游戏界面
4. 使用行 `开始学习.bat` 根据程序提示进行操作

## ❗其他说明

1. 本程序基于日服进行开发，但通过自定义配置和图片资源截取，理论上也可在其它服游戏端使用部分功能，欢迎二次开发
